RAGMind

Intelligent B2B SaaS Knowledge & Support Platform

Turning company documents into searchable project knowledge bases and Telegram customer-support bots.

Business Value

  • Centralized Knowledge: Upload and manage company documents effortlessly.
  • Instant Answers: RAG pipeline ensures accurate, context-aware answers to user queries.
  • Automated Customer Support: Deploy intelligent Telegram bots powered by your company's proprietary data.
  • Multi-tenant SaaS: Role-based access (Company Admin vs. Platform Owner) with strict data isolation per project.

Key Features

Knowledge Management

  • Document Upload & Processing
  • Intelligent Context Retrieval
  • Dashboard Testing Environment
  • Project Scoping & Analytics

Telegram Integrations

  • Multiple Bot Integrations
  • Durable Conversations
  • Source Visibility Control
  • Human-Handoff Settings

Architecture Overview

  • Frontend: Next.js App Router (React)
  • Backend: FastAPI + SQLAlchemy (async, Python)
  • Background Jobs: Celery + RabbitMQ + Redis
  • Database: PostgreSQL + pgvector (Optional: Qdrant)

Level 1: AI & RAG Pipeline

  • Extraction: Celery workers extract text from files securely.
  • Chunking: Content is split into semantic chunks.
  • Embeddings: Generated via providers (Gemini, Cohere).
  • Generation: Vector search retrieves chunks; LLMs (Groq, OpenRouter, Cerebras) generate context-aware answers.
LLMs Embeddings Vector DB Celery Workers

Level 2: Database Layer

  • Relational Data: PostgreSQL reliably stores Users, Projects, Assets, Telegram Customers, and Conversations.
  • Vector Storage: pgvector stores document chunk embeddings natively in PostgreSQL.
  • Migrations: Managed by Alembic to ensure safe schema evolution.
  • Vector Alternatives: Built-in support for Qdrant as an optional vector backend.

Level 3: Integrations & Bots

  • Telegram Webhooks: Registers bots securely to dynamic HTTPS backend URLs.
  • Conversations: Persists customer messages and orchestrates RAG specific to the company's project.
  • Outbox Pattern: Responses are queued as pending and delivered by a reliable background worker, preventing lost messages.
  • Token Encryption: Bot tokens are encrypted via Fernet and are never returned to clients.

Level 4: Infrastructure & Monitoring

  • Local Stack: Docker Compose for seamless, single-command dev setup.
  • Observability: Prometheus & Grafana with pre-configured dashboards.
  • Metrics: Complete monitoring of FastAPI endpoints, Celery workers, and Node.
  • Cloud Production: Automated deployment to Azure Container Apps (ACR, PostgreSQL Flexible Server).

Conclusion

RAGMind offers a scalable, secure, and modern architecture connecting a Next.js UI, FastAPI logic, and powerful LLM capabilities to deliver real business impact.
